# TASK_20260625_003 vibego 系统架构图多格式对比

## 1. 任务目标

用户希望同一张 vibego 系统架构图同时以多种方式交付，用于比较 HTML、PNG、SVG、Mermaid 哪种在 Codex 内预览与沟通效果最好。

本次产物：

- `TASK_20260625_003_vibego系统架构图.html`：单文件 HTML，可直接用 Codex / 浏览器打开。
- `TASK_20260625_003_vibego系统架构图.svg`：独立矢量图。
- `TASK_20260625_003_vibego系统架构图.png`：由 SVG 导出的纯架构 PNG，便于聊天里快速查看。
- `TASK_20260625_003_vibego系统架构图_HTML预览.png`：由单页 HTML 导出的预览截图，用于比较 HTML 页面整体效果。
- `TASK_20260625_003_vibego系统架构图.mmd`：Mermaid 源码，便于后续维护。
- `TASK_20260625_003_vibego系统架构图.md`：本任务说明、证据、验证记录。

## 2. 设计输出

### visual thesis

用“控制面板式分层地图”表达 Telegram 到本地模型 CLI 的编排关系，克制浅色、少装饰，优先可读性和证据可追踪。

### content plan

1. 总览图：入口层、控制面、Worker 面、运行面、存储目录。
2. 对比说明：HTML、SVG、PNG、Mermaid 各自优劣。
3. 证据锚点：每个架构事实对应仓库文件与锚点。
4. 维护建议：后续复杂逻辑优先 Mermaid/HTML，沟通确认补 PNG。

### interaction thesis

HTML 只提供页内跳转、横向滚动、证据折叠和节点 hover/focus 高亮；不使用复杂动效，避免影响架构阅读。
