# /TASK_0085 任务列表增加显示任务编码前缀，拼接在任务状态和任务名称的中间

## 1. 背景

用户确认需求：

1. 任务按钮文案在 **状态图标** 与 **任务名称** 中间增加 **任务编码前缀**
2. 任务编码显示格式：**`TASK_0001`**
3. 影响范围：**任务列表 + 任务搜索结果**

目标样式示例：

```text
🔍 TASK_0001 调研任务
🧪 TASK_0002 修复登录问题
✅ TASK_0003 发布上线
```

- 现有列表视觉基线：`docs/TASK_0048_去除任务列表中的任务类型图标.md`

## 6. 最小实现

### 6.1 按钮文案生成

- `bot.py::_compose_task_button_label(...)`
  - 从：

```text
状态图标 + 标题
```

  - 调整为：

```text
状态图标 + TASK_编码 + 标题
```

- 若无状态图标：

```text
TASK_编码 + 标题
```

### 6.2 截断策略

- 保留：
  - 状态图标
  - 任务编码
  - 运行中后缀 `▶️`（若存在）
- 允许标题部分被截断并追加 `…`

### 6.3 影响入口

- `_build_task_list_view(...)`
- `_build_task_search_view(...)`

由于这两处都复用 `_compose_task_button_label(...)`，无需额外分叉逻辑。
