03形态判断

当前图型能否表达主谓宾？

不能表达“谁作用于谁 / 什么导致什么 / 数据从哪到哪 / 状态如何转移”时，禁止靠标题硬命名，必须换图型。

04布局算法门禁

先分配画布

宽度放泳道、参与者列、before/after；高度放时间、阶段、因果递进和证据展开。

先节点后连线

节点尺寸、间距、旁注位置先确定；箭头只连边缘锚点或时序生命线中心。

失败即重排

节点重叠、线穿字、文字溢出、390px 不可读，任一出现都不能交付。

05验证闭环

测试锁定规则

新增 shape contract、layout/arrow/collision 两个测试，并扩展 AGENTS 注入测试，确保 pipx/AGENTS 同步后仍保留规则。

节点不覆盖同层统一留白，最小间距 16px，主路径阶段 28px。
箭头不穿字连线层低于节点层；标签放线段中点或拐点旁。
文字不溢出HTML/CSS 可换行容器，max-width、min-height、line-height、overflow-wrap:anywhere。
双宽度检查桌面检查宽度利用与锚点；390px 检查纵向降级与无横向依赖。
