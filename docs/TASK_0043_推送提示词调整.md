# /TASK_0043 推送提示词调整

## 背景

用户希望在“🧩 部分按推荐（需补充）”快捷回复场景中，推送给模型的提示词不再包含旧的三行模板，而是使用更简洁的固定前缀。

## 目标（用户口径）

当用户走“部分按推荐（需补充）”并输入补充说明时，推送给模型的提示词必须以以下两行开头：

```text
未提及的决策项全部按推荐。
用户补充说明：
```

并在后续追加用户的补充说明内容。

## 变更点（已实现✅）

- ✅ 更新 `bot.py` 中 `_build_quick_reply_partial_prompt()` 的输出格式：
    - 固定前缀改为：
        - `未提及的决策项全部按推荐。`
        - `用户补充说明：`
    - 补充说明内容作为下一行追加（为空时不追加，最终仅保留两行前缀）。
- ✅ 更新回归测试断言：`tests/test_model_quick_reply.py`
- ✅ 同步更新相关说明文档：`docs/TASK_0041_模型回复底部快捷回复按钮.md`

## 自测方法

运行：

```bash
pytest -q tests/test_model_quick_reply.py
```

验证点：

- 输入补充说明时，推送 prompt 含两行固定前缀 + 原始补充说明内容。
- 输入“跳过/空消息”时，仍推送 `待决策项全部按模型推荐`（不受本次修改影响）。
