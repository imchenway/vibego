## 实现

1. `agents-sync` 接受零内置 skill 的源目录，仍生成有效的空 skills override。

3. `scripts/run_bot.sh` 与 `scripts/start_tmux_codex.sh` 不再把空 skills 目录判定为 override 损坏。
