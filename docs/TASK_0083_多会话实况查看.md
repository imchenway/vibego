# /TASK_0083 多会话实况查看

## 1. 背景

用户确认需求：

1. 任务列表中，已有会话在运行的任务，标题末尾显示一个运行中的小图标
2. `终端实况` 按钮改名为 `会话实况`
3. 点击后先展示当前项目的 tmux 会话列表
4. 会话范围：**主会话 + 活动并行**
5. 运行中小图标：**`▶️`**

## 2. 证据

- 主键盘按钮：`bot.py`（锚点：`WORKER_TERMINAL_SNAPSHOT_BUTTON_TEXT`, `_build_worker_main_keyboard`）
- 旧“终端实况”直接读取默认主会话：`bot.py`（锚点：`_handle_terminal_snapshot_request`, `_capture_tmux_recent_lines`）
- 任务列表按钮文案：`bot.py`（锚点：`_build_task_list_view`, `_compose_task_button_label`）
- 并行会话持久化：`parallel_runtime.py`（锚点：`ParallelSessionRecord`, `ParallelSessionStore`）
- 并行会话健康校验：`bot.py`（锚点：`_parallel_session_runtime_issue`, `_get_active_parallel_session_for_task`）

## 5. TDD 红灯

先补测试，覆盖：

1. 主键盘按钮改为 `💻 会话实况`
2. 点击入口先展示会话列表，而不是直接抓默认 tmux
3. 任务标题命中活动会话时，末尾追加 `▶️`
4. 会话列表包含主会话与并行会话
5. 并行会话详情应命中对应 `tmux_session`
6. stale 并行会话应自动降级并从活动列表剔除

首次执行：

```bash
python3.11 -m pytest -q tests/test_chat_menu_buttons.py tests/test_task_list_entry.py tests/test_session_live_view.py -k "worker_keyboard_button_text or worker_session_live_button_opens_session_list or compose_task_button_label_appends_running_suffix or build_task_list_view_marks_running_tasks or session_live"
```

结果：

- ❌ `6 failed`
- 失败原因：
  - 按钮文案仍是 `💻 终端实况`
  - 入口仍直接截取默认 tmux
  - 任务列表按钮不支持 `is_session_running`
  - 缺少 `SessionLiveEntry` / 会话列表 / 会话详情能力

## 6. 最小实现

### 6.1 主键盘与入口升级

- `WORKER_TERMINAL_SNAPSHOT_BUTTON_TEXT` 改为 `💻 会话实况`
- `on_tmux_snapshot_button(...)` 继续复用原入口，但行为改为：
  - 先构造并发送“会话列表页”
  - 不再直接抓取默认主 tmux

### 6.2 任务列表运行中图标

- `_compose_task_button_label(...)` 增加 `is_session_running`
- `_build_task_list_view(...)` / `_build_task_search_view(...)`
  - 先汇总活动任务集合
  - 命中时在标题末尾追加 `▶️`

### 6.3 多会话聚合

- 新增 `SessionLiveEntry`
- 新增：
  - `_list_native_active_task_ids()`
  - `_list_active_parallel_sessions()`
  - `_list_project_live_sessions()`
- 口径：
  - 主会话始终展示
  - 并行会话只展示未 `closed/deleted` 且 runtime 健康的记录
  - 批量扫描中发现 stale，会自动 `closed` 并清理绑定

### 6.4 会话列表与会话详情

- 新增：
  - `_build_session_live_list_view()`
  - `_build_session_live_snapshot_view(entry_key)`
  - 会话列表 / 会话详情的 inline callbacks
- 详情页按钮：
  - `🔄 刷新当前会话`
  - `⬅️ 返回会话列表`

### 6.5 并行会话批量查询

- `parallel_runtime.py::ParallelSessionStore.list_sessions(...)`
  - 支持按当前项目列举会话
  - 支持按状态过滤
