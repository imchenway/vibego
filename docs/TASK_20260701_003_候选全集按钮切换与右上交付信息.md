## 3. 交互口径

- 候选按钮使用 `role="tablist"` / `role="tab"` / `aria-selected` / `aria-controls`。
- 候选面板使用 `role="tabpanel"` / `aria-labelledby`。
- 默认显示首选候选 A。
- 支持 `#candidate-b` 这类 URL hash 深链直接打开对应候选。
- 支持键盘左右方向键切换候选，`Home` / `End` 跳转首尾。
- HTML 初始状态不写死 `hidden`，脚本成功后才隐藏非当前候选；无 JS 时全部候选纵向展开。
